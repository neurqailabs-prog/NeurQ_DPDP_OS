========================================
QS-DPDP OS - Complete Installer
========================================

INSTALLER FILE: QS-DPDP-OS-Installer.bat

This is a complete installer that will install the entire
QS-DPDP OS solution on your system automatically.

========================================
HOW TO USE
========================================

SINGLE FILE INSTALLATION:

1. Run: QS-DPDP-OS-Installer.bat
2. Press any key to start installation
3. Wait for installation to complete
4. Done! Application is installed

========================================
WHAT IT INSTALLS
========================================

✅ QS-DPDP OS Application
✅ Desktop Shortcut (with custom icon)
✅ Start Menu Entry
✅ Splash Screen
✅ Documentation
✅ Configuration Files
✅ Uninstaller

========================================
INSTALLATION LOCATION
========================================

C:\Users\[YourUsername]\AppData\Local\NeurQ\QS-DPDP-OS

This location doesn't require administrator privileges.

========================================
AFTER INSTALLATION
========================================

To run QS-DPDP OS:
  1. Double-click desktop shortcut "QS-DPDP OS"
  2. Or use Start Menu: NeurQ > QS-DPDP OS

The application will:
  - Show Visual Studio-style splash screen
  - Launch main application
  - Display RAG AI and Quantum-Safe theme

========================================
UNINSTALLATION
========================================

To uninstall:
  Run: C:\Users\[YourUsername]\AppData\Local\NeurQ\QS-DPDP-OS\uninstall.bat

Or manually:
  1. Delete desktop shortcut
  2. Delete Start Menu entry
  3. Delete installation directory

========================================
INSTALLER PACKAGE
========================================

For distribution, use:
  QS-DPDP-OS-Installer-Package.zip

This contains everything needed for installation.

To install on another system:
  1. Extract the ZIP
  2. Run: QS-DPDP-OS-Installer.bat
  3. Done!

========================================
REQUIREMENTS
========================================

- Windows 10/11
- Java 21+ (for running application)
- No administrator privileges needed
